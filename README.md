# Circular Progress & Pie Chart (React)

## Проект, реализующий два независимых компонента:

1. Круговой прогресс-бар на SVG (без сторонних библиотек)
2. Круговая диаграмма (Pie Chart) с возможностью управления секторами

### Стек: React, TypeScript, Chart.js, CSS Modules, react-colorful

### Функциональность
Progress Bar:
- SVG-реализация
- Плавная анимация
- Состояния: in progress / success / warning / error
- Градиент от красного к зелёному
- Режим dashboard

Pie Chart:
- Добавление секторов через форму
- Выбор цвета (color picker)
- Динамическое обновление

### В разработке
- Сохранение данных в localStorage
- Редактирование и удаление секторов
- Drag&Drop сортировка
- Экспорт диаграммы в PNG
- Тёмная тема
